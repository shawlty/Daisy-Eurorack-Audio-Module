# WaveForge-1

Real-Time Embedded Audio DSP Platform built with Daisy Seed.

## Features

* Stereo Audio Processing
* Real-Time Audio Callback
* OLED Interface
* Rotary Encoder Controls
* Potentiometer Inputs
* Eurorack Expansion
* Modular Firmware

## Hardware

* Daisy Seed
* OLED I2C Display
* TL072 Audio Stage
* LM2596 Power
* EC11 Encoder

## Firmware

C++

libDaisy

STM32H750
